在Javascript中，new之后发生了什么？

Mar 26, 2018 | JavaScript

当初学习Javascript的时候，读的是《Eloquent JavaScript》，看到new出现，完全无法理解，这个操作符是做什么的，后来看了别的资料，才慢慢理解了。

其实这个还是比较简单的，通过new会完成下面5件事情：

1.它会创建一个新的对象。

2.它会让这个新对象的__proto__属性指向构造函数的prototype，也就说指向它的原型。

3.它会让构造函数的this指向这个新对象。

4.它会执行这个构造函数，并且执行的时候，函数里面的this都是指向这个新对象。

5.如果构造函数有返回值，而且里面的返回值是基本数据类型，那么就放弃这个值，返回新对象，但如果返回值是引用类型，那么就返回这个引用类型的对象。
